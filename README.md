# Cazador de Tesoros

## Descripción
Explora ub tablero misterioso para encontrar tesoros ocultos mientras evitas trampas peligrosas. Cada movimiento cuenta y el tiempo corre. ¿Podrás encontrar todos los tesoros?

## Características
- **Interfaz gráfica intuitiva** con Java Swing
- **Sistema de Puntuación** basado en tiempo y movimientos
- **Diferentes niveles de dificultad** (Fácil, Medio, Difícil)
- **Indicadores de Proximidad** a tesoros
- **Tablero dinámico** con generaciòn aleatoria
- **Sistema de records** persistentes

## Cómo Jugar
1. **Inicia el juego** seleccionando una dificultad
2. **Haz clic** en las celdas para descubrir su contenido
3. **Los números** indican cuantos tesoros hay en las celdas adyacentes
4. **Encuentra todos los tesoros** para ganar
5. **Evita las trampas** - ¡pierdes al caer en una!

## Controles
- **Clic Izquierdo**: Revelar Celda
- **Botón Reiniciar**: Comenzar nueva partida
- **Botón Menú**: Volver al menú principal

## Requisitos del Sistema
- Java 8 o superior
- 100 MB de espacio libre
- Resolución mínima 800x600
